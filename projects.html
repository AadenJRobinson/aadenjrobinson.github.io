Projects

The projects I currently do are purely hobby projects, and are not intended to benefit me financially in the future. The following project is a programming one that I have been working on for about a month now. It is intended to be a world generation algorithm for the game Minecraft that allows for the creation of floating islands that are created as the player travels. I currently have two concepts for the algorithm, both with their own set of benefits and drawbacks. I am trying to figure out how to merge them together for the best of both worlds, and when I do, a much better world generation algorithm will be the result.

My secondary project, which is "complete", is an addon to a bending plugin called ProjectKorra, which attempts to recreate the idea of elemental bending (from Avatar: The Last Airbender) in the game Minecraft. This addon includes multiple moves, which have yet to be released on the server I made them for.  I plan on adding more to this project as it is entertaining as well as good for the server.

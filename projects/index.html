Projects

Ether Infinity Skyblock

This is a world generator and manager designed for the Minecraft: Java Edition server EtherealCraft. This software procedurally generates worlds with floating islands (commonly known as "Skyblock" islands) spread out randomly, and of varying sizes. The user can select create multiple worlds, each from one of four size presets. All the necessary data is stored in a database when the server shuts down, and virtually every setting is configurable. I hope to complete this project and publish it to the server before I graduate.

Titans of CNC

The Titans of CNC are parts that we design on SolidWorks, create toolpaths for with MasterCam, and finally cut from aluminum stock metal using Haas milling machines. Scroll or click through the images below to see more.

Senior Project

This year, I am working on a senior design project where I come up with an idea, present it to my class, design a physical representation in SolidWorks, and eventually make toolpaths for it via Mastercam and cut it out of physical aluminum with Haas CNC machines (and CNC lathes in my case).

For my project, my group and I designed a machinist's "speed" vise handle, which has two locations to tighten a CNC vise – one on the end, for leverage; and one in the middle, for speed. The design has been 3D modelled and drawn with SolidWorks, as well as 3D printed.
